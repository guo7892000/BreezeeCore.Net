字符分隔拼接

一、功能概述

实现对分隔后的字符多次接拼！典型的例子为表中所有列的字符常量设置（常量名与值都包含列名）。

三、操作步骤

1、【要分隔的字符（集合）】：录入要分隔的原始字符信息。
2、界面元素输入或选择：
【分隔符列表】文本框：字符是以哪些字符来分隔的，以逗号分隔多个分隔符
【分隔符列表的分隔符】文本框：分隔符列表中的多个分隔符是用哪个字符来分隔的，默认为逗号。
【单行汇总成一列】：选中时，如数据只有一行时，分隔后所有值都在A列中。
【忽略分隔后的空数据】：选中时，分隔的空字符将被去掉，即列值会前移。
【每项剔除前后空白字符】：选中时，针对分隔后的字符再剔除前后空白字符。
【结果换行】：选中时，输出结果会换行。
【仅根据空白字符分隔】：选中时，就只根据空白字符来分隔数据。
【示例】：可选。当选中时，会自动加载示例数据，方便了解本功能的使用。
3、点击【分拆字符】：根据分隔符和分隔选项，对字符分隔，并加载到【分拆后结果】网格中。
4、要拼接的字符格式（输入）：录入要输出的内容格式，其中#列名#作为【分拆后结果】网格对应列内容。
5、点击【转换】：转换成功后，会显示在【转换后的字符（输出）】文本框内。
