文件字符替换

一、功能概述

批量替换文件字符，如将文件中的“旧文本A”字符替换为“新文本B”。可以选择本地文件或文件夹，也支持从FTP服务下载文件来替换并上传。本功能的FTP服务器配置只能保存一个。

三、操作步骤

1、选择文件来源：
本地文件：要替换的字符来源于本地文件。
FTP远程文件：需要从FTP服务器下载文件到本地。
2、选择【最终生成目录】：最终替换文件后的保存目录。
3、选择【文件字符集】：文件的字符集格式，通常为UTF-8。
4、选择源文件来复制
方式一：文件来源为【本地文件】时：
右击【待复制的本地源目录或源文件】，选择本地要复制到【最终生成目录】的文件夹或文件。
其中排除文件和排除文件夹，用于排除选中目录或文件。暂未实现选择本地目录时，根据排除配置过滤目录和文件
方式二：文件来源为【FTP远程文件】时： 录入FTP服务器相关信息，包括：
服务器配置：服务器IP、端口、用户、密码、协议、默认目录、连接字符集。
下载配置：读取目录、保存本地路径、下载路径不含读取目录、下载前清空目录、排除文件、排除目录、自动排除，操作包括：下载所有文件、显示所有文件、下载所选文件
上传配置：上传目录、备份目录、上传方式（先删后增、直接覆盖、备份后覆盖）。操作包括：上传目录、上传所选文件
5、录入要替换的新旧字符：替换的新旧字符网格中录入新旧字符，或通过Excel复制粘贴新旧字符。
6、点击【复制并替换】：将文件复制到【最终生成目录】，并根据新旧字符来替换。
如【最终生成目录】是自己通过其他软件下载文件的，则可以直接点击【仅字符替换】，则直接根据新旧字符来替换。
替换成功后，会直接显示处理结果页签。
7、上传文件到FTP服务器【可选】：我们可以直接点击【上传目录】，把结果全部上传到FTP服务器上。
也可以在网格中勾中要上传的文件，然后点击【上传所选文件】，把所选文件上传到FTP服务器上。

保存路径：复制结果的保存路径。【只复制选中】复选框指定只复制选中的文件。 查找文件清单网格：可以录入或粘贴要找的文件名。注：这里的文件名是精确匹配！ 排除文件名：方便排除结果中不需要的文件【文件名包含时排除，多项以逗号分隔】。【指定文件后缀】选中时，可只复制提定后缀文件名的。 点击【过滤】，即可根据录入的文件名网格及排除文件名、后缀等，确定要找的文件并选中。 点击【复制】，就可以把结果网格中选中的文件复制到【保存路径】中。
