目录字符生成

一、功能概述

针对选择的目录，得到文件、目录或包括两者的清单字符信息。

三、操作步骤

1、界面元素输入或选择：
【目录选择】：必填，我们要对哪个目录进行搜索。
【输出类型】：包括【仅文件】、【仅目录】、【目录与文件】。
【路径类型】：包括【全路径】、【仅文件名】、【相对路径】。
【是否指定目录深度】：【目录深度】前面的复选框，默认不选中，当选中时，输入的【目录深度】才有效。
【目录深度】：搜索到第几层目录，根目录为第一层。
【仅显示文件目录】：复选框，只针对文件。应用场景：“把文件复制到当前批处理文件所在目录的相对路径下”中目标目录的内容！
前缀：字符前面插入的内容。
后缀：字符后面追加的内容。
替换前：要被替换的旧字符。
替换后：取代旧字符的新字符。
2、点击【生成字符】：生成成功后，会提示相关信息。

四、逻辑与业务规则

关于【路径类型】： 【全路径】是包括盘符的文件或目录全路径；【仅文件名】即指文件或目录名；【相对路径】是指相对当前选择目录的相对路径。
