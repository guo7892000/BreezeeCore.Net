两个Excel合并

一、功能概述

将两个有关联条件的Excel网格内容，关联合并为一个网格。

三、操作步骤

1、复制粘贴数据1：在Excel复制一列数据，并粘贴到网格Excel1上。
2、复制粘贴数据2：在Excel复制一列数据，并粘贴到网格Excel2上。
3、【合并类型】：包括交集、左集、右集、全集。
4、修改【关联条件】输入框里的信息为Excel1和Excel2关联的条件，格式为：A=A1。其中A为Excel1中的列名，A1为Excel2中的列名。
4、点击【生成】：
生成成功后，会提示相关信息。

四、逻辑与业务规则

关于从Excel中复制与粘贴： 复制的第一行必须为列头信息，否则第一行数据会被过滤掉。在网格中按Ctrl + v 即可粘贴数据。
