左右交换字符

一、功能概述

通过分隔字符，将其两边的数据交换后，再进行自定义字符替换，并在最后追加指定字符。常见的应用场景为控件的赋值与取值。

三、操作步骤

1、界面元素输入或选择：
分隔符：字符是以哪些字符来分隔的。
要转换的字符：需要交换位置的字符，可以是多行。
替换前旧字符：要被替换的旧字符。
替换后新字符：取代旧字符的新字符。
结尾附加的字符：每行最后追加的字符。
加载示例数据：复选框，当选中时会自动加载样式数据，方便我们了解该功能。
6、点击【转换】：转换成功后，会提示相关信息。

四、逻辑与业务规则

分隔符：可以是单个字符或一段字符串。
替换后新字符可以为空，相当于直接去掉旧字符。
