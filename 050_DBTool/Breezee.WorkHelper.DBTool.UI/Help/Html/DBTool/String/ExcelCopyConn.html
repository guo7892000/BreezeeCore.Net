复制数据拼接

一、功能概述

将粘贴进来的数据拼接成自定义格式的字符串，或UNION清单，或UNION ALL清单。

二、界面截图

主界面：操作文件生成

三、操作步骤

1、复制粘贴数据：在Excel复制一列数据，并粘贴到网格上。
2、界面元素输入或选择：
【语句类型】：包括【自定义拼接字符串】、【UNION清单】、【UNION ALL清单】。
【数据库类型】：包括五种数据库。只有当【语句类型】为【UNION清单】、【UNION ALL清单】时才显示。
【自动字母列名】：可选，选中时列名显示为字母。
【拼接字符】：文本框，输入要拼接字符的格式。请用#列名#作为参数配置！
3、点击【生成】：生成成功后，会提示相关信息。

四、逻辑和业务规则

关于参数： 请用#列名#作为参数配置。我们也可以在网格中右击选择【加入】，将参数添加到拼接字符中。
