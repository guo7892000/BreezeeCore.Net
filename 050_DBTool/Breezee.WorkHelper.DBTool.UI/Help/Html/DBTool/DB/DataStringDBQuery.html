数据库数据拼接

一、功能概述

通过查询数据库某个表的数据，然后将其某些列数据拼接起来。

三、操作步骤

1、选择【连接配置名】：要连接哪个数据库。
2、输入【选项配置】相关信息：
表名：下拉框，可以手工输入；也可以在勾中【获取表清单】复选框后下拉选择。
获取表清单：复选框，当选中时会把表清单加载到【表名】下拉框中。
Where条件：文本框，可输入筛选数据的条件，生成部分数据SQL。
拼接字符：文本框，输入要拼接字符的格式。请用#列名#作为参数配置！
3、点击【生成】：成功后会提示相应信息。

四、逻辑与业务规则

【Where条件】： 如表名不为空，那么Where条件作为表的过滤条件；如果表名为空，那么where条件可输入自定的SQL语句！
