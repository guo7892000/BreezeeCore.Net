三、操作步骤

【数据库工具DB设置】：点击按钮，即打开数据库工具的数据库配置窗体，可以修改数据库工具模块所连接的数据库。注：请使用SQLite数据库文件配置！
具体操作步骤：复制程序所在目录的数据库文件【SQLite_DBTool.db】，然后粘贴到非系统分区目录。然后使用该窗体设置使用SQLite数据库类型，并指向该文件，然后退出应用并重新登录即可。
【加密】：可以将输入的字符，加密为数据库工具模块所使用的密文。

四、逻辑与业务规则

主框架的启动配置文件路径： C:\Users\用户目录\AppData\Roaming\WorkHelper\StartUp.xml
注：【用户目录】为操作系统的登录用户名！
其中【ConfigPath】配置项为本工具的【个性化配置文件路径】，而【数据库工具DB设置】的配置文件相对该目录的子目录下【\Config\DB\DbConfigDBTool.xml】
