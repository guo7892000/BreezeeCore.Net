表列扩展信息SQL生成

一、功能概述

为了更好地理解数据库表和列的含义，我们可以对其增加一些备注信息。但因为各种数据库差异，可能只有部分数据库类型支持。例如SQLite就不支持！

三、操作步骤

1、【模板下载】：下载后维护相关信息。
表名：必填，针对哪个表。 列名：非必填。当其为空时，是针对表的注释；非空时是针对列的注释。 扩展属性说明：具体的注释内容。 2、点击【导入】：选择刚刚维护的模板文件，点击【打开】即可将结果导入到网格中。
3、选择【数据库类型】，包括SqlServer、Oracle、MySql、SQLite、PostgreSql
4、点击【生成】：
生成成功后，会提示相关信息。
