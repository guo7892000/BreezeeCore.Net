实体类生成

一、功能概述

针对数据库中的表、视图，通过模板导入生成实体对象。模板样式可以自定义，也可以复用系统参数，构造自定义的参数！
生成思路：通过读取自定义模板，然后取出里边的参数（格式：#参数#），根据每个表和导入模板中的参数信息，替换参数为具体值，从而得到实体类代码。

三、操作步骤

1、【模板下载】：下载后维护相关信息。
2、选择【连接配置名】：从下拉框中选择，会自动加载数据库清单。数据来源于【数据库连接配置】功能。
3、点击【导入】：选择刚刚维护的模板文件，点击【打开】即可将结果导入到网格中。
4、选择模板：导入的所有模板会显示在下拉框中，可选择更换。
5、展开树，选择子节点：

全部生成：如果选择的是数据库节点，则右击选【全部生成】，可生成多个表的实体类文件。
单表生成：如果选择的是表节点，则右击选【单表生成】，可生成该表的实体类文件。
