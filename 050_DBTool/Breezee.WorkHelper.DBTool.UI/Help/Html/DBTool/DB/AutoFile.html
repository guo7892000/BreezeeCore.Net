一、功能概述

针对新增的功能，因为其包含UI、接口、实现层等相关文件，如果手工一个个新建则比较花时间。本功能就是利用模板实现这些文件的批量创建。

三、操作步骤

1、【模板下载】：下载后维护相关信息。
2、确认或修改【类模板】下各页签信息：可以修改各模板的内容。
3、点击【导入】：选择刚刚维护的模板文件，点击【打开】即可将结果导入到网格中。
4、输入或选择本次生成的信息： 【生成的根路径】：必填
【空间名加上目录】：可选。
6、点击【生成】：
生成成功后，会提示相关信息。我们就可以到【生成的根路径】中查看所有文件。

四、逻辑与业务规则

【空间名加上目录】复选框： 当选中时，类的空间名会加上目录名；否则不会。
